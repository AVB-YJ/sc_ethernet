For initial development of ethernet applications the following XMOS development platforms can be used:

  • XC-2 Ethernet Kit
  • XDK XS1-G Development Kit
  • XP-DSC-BLDC Motor Control Platform
  • XK-AVB-LC-SYS AVB Audio Endpoint
  • XC-3 Dual-port Ethernet Kit

For developing an application specific board with ethernet please refer to the hardware guides for the above boards with example schematics, BOMs, design guidelines etc.
